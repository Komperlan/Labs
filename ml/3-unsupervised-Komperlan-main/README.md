# Лабораторная 3. Обучение без учителя

Проект решает задачу кластеризации автомобилей по их характеристикам. Используется тот же датасет объявлений, что и в лабораторной работе по обучению с учителем, но целевая переменная не предсказывается: цель состоит в поиске групп похожих автомобилей.

## Основной файл

- `lab3.ipynb` - ноутбук с предобработкой данных, baseline-кластеризацией, подбором K-Means и сравнением с собственной реализацией.

## Входные данные

Ноутбук ожидает файл:

```text
./data/cars_data.csv
```

CSV можно получить из проекта `1-parser-Komperlan-main`. Для воспроизведения нужно положить файл в папку `data` внутри проекта или изменить путь в ноутбуке.

## Предобработка

В ноутбуке выполняются следующие действия:

- удаление дубликатов;
- заполнение пропусков в пробеге;
- выделение марки и модели из `title`;
- очистка цвета;
- удаление выбросов по цене и мощности двигателя;
- one-hot encoding категориальных признаков;
- стандартизация признаков через `StandardScaler`.

## Модели и подходы

- Baseline: разбиение автомобилей на кластеры по квантилям цены.
- Sklearn K-Means: библиотечная реализация `KMeans` с подбором гиперпараметров.
- Custom K-Means: собственная реализация K-Means с `random` и `k-means++` инициализацией, несколькими запусками и контролем сходимости.

## Метрики

Используются:

- `Silhouette Score`;
- `Calinski-Harabasz Index`;
- `Davies-Bouldin Index`;
- `Inertia`.

## Результаты

Baseline по цене:

- `Silhouette Score`: `-0.0944`;
- `Calinski-Harabasz`: `7.61`;
- `Davies-Bouldin`: `10.4380`.

Sklearn K-Means:

- лучшие параметры: `n_clusters=6`, `init=k-means++`, `max_iter=100`, `n_init=20`, `tol=1e-05`;
- `Silhouette Score`: `0.1800`;
- `Calinski-Harabasz`: `9.81`;
- `Davies-Bouldin`: `3.8065`;
- `Inertia`: `579056.11`.

Custom K-Means:

- лучшие параметры: `n_clusters=4`, `init_method=k-means++`, `max_iter=100`, `n_init=20`, `tol=0.0001`;
- лучший `Silhouette Score`: `0.2148`;
- до сходимости потребовалось `12` итераций.

Вывод: обе полноценные реализации K-Means заметно лучше baseline. Собственная реализация получила лучший `Silhouette Score`, а sklearn-реализация оказалась сильнее по `Davies-Bouldin` и `Inertia`.

## Визуализация

Для визуального сравнения кластеров используется PCA-проекция в 2D. В ноутбуке также построены графики распределений числовых признаков и сравнение метрик.
