# Лабораторная 1. Парсер объявлений Auto.ru

Проект собирает датасет объявлений о продаже автомобилей с Auto.ru для дальнейших лабораторных работ по машинному обучению. Парсер проходит по страницам выдачи Auto.ru для Санкт-Петербурга, извлекает базовые характеристики автомобилей и сохраняет результат в CSV.

## Что делает проект

- Загружает страницы выдачи Auto.ru по URL из `config.py`.
- Парсит название автомобиля, цвет/комплектацию, год выпуска, пробег, цену и технические характеристики.
- Разделяет строку двигателя на объем в литрах и мощность в лошадиных силах.
- Очищает числовые поля от пробелов, неразрывных пробелов и текстовых суффиксов.
- Сохраняет итоговую таблицу в `cars_data.csv`.

## Структура

- `main.py` - основной скрипт парсинга и подготовки CSV.
- `config.py` - URL шаблон страницы выдачи Auto.ru.
- `cars_data_example.csv` - пример уже собранных данных.
- `LICENSE` - лицензия проекта.

## Данные

В итоговом CSV используются следующие признаки:

- `title` - марка, модель и описание объявления.
- `color` - цвет или строка с комплектацией и цветом.
- `year` - год выпуска.
- `mileage` - пробег в километрах.
- `price` - цена в рублях.
- `body` - тип кузова.
- `wd` - тип привода.
- `transmission` - коробка передач.
- `engine_volume_l` - объем двигателя.
- `engine_power_hp` - мощность двигателя.

## Запуск

```bash
pip install requests beautifulsoup4 pandas
python main.py
```

После запуска рядом со скриптом появится файл `cars_data.csv`.

## Важные замечания

Парсер завязан на HTML-классы Auto.ru. Если сайт изменит верстку или начнет активнее блокировать запросы, селекторы в `main.py` нужно будет обновить. Скрипт также делает много запросов подряд, поэтому для стабильной работы может понадобиться ограничение скорости запросов.
