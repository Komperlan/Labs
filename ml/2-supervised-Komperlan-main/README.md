# Лабораторная 2. Обучение с учителем

Проект решает задачу регрессии: по характеристикам автомобиля нужно предсказать его цену. В качестве данных используется CSV с объявлениями о продаже автомобилей, подготовленный в лабораторной работе с парсером.

## Основной файл

- `lab2.ipynb` - ноутбук с загрузкой данных, предобработкой, визуализацией, обучением моделей и сравнением результатов.

## Входные данные

Ноутбук ожидает файл:

```text
./data/cars_data.csv
```

Если данные получены из проекта `1-parser-Komperlan-main`, файл `cars_data.csv` нужно положить в папку `data` внутри этого проекта или изменить путь в ноутбуке.

## Предобработка

В ноутбуке выполняются следующие шаги:

- удаление дубликатов;
- заполнение пропусков в пробеге нулями;
- выделение `brand` и `model` из поля `title`;
- очистка цвета от части с комплектацией;
- удаление выбросов по цене и мощности двигателя через IQR;
- one-hot encoding категориальных признаков.

## Модели

Рассматриваются три подхода:

- baseline-модель, которая всегда предсказывает среднюю цену;
- `HistGradientBoostingRegressor` из scikit-learn с подбором гиперпараметров через `GridSearchCV`;
- собственная реализация градиентного бустинга `SimpleGradientBoosting` на деревьях решений.

## Результаты

Baseline:

- средняя цена: `2 724 399` руб.;
- RMSE: `2 286 534` руб.

Лучшая sklearn-модель:

- модель: `HistGradientBoostingRegressor`;
- параметры: `learning_rate=0.01`, `max_depth=2`, `max_iter=400`, `max_leaf_nodes=4`;
- train RMSE: `1 192 890`, train R2: `0.768`;
- test RMSE: `1 002 487`, test R2: `0.704`.

Собственная модель:

- параметры: `n_estimators=180`, `learning_rate=0.01`, `max_depth=9`, `min_samples_leaf=10`, `tol=5`;
- validation RMSE: `797 040`, validation R2: `0.853`;
- test RMSE: `1 054 020`, test R2: `0.673`.

Вывод: библиотечная модель лучше обобщается на тестовой выборке, хотя собственная реализация показывает близкий результат.
